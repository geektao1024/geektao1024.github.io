Starter Lab

This is a starter tool page.

Use this folder as a clean starting point for any new static tool you want to publish under the studio site.

How to Build a New Tool

Copy the folder

Duplicate tools/starter/ and rename it for the new tool.

Replace the content

Swap the placeholder title, body copy, scripts, and assets with the real product.

Add a homepage link

Link the new tool from the homepage so visitors and search engines can discover it.
